Cisco IOS config reference
Paste on each device. Replace 10.100.0.50 with your monitoring host IP.
ip access-list standard ACL_SNMP_NETWATCH
  permit 10.100.0.50
  deny   any log

snmp-server community netwatch_ro ro ACL_SNMP_NETWATCH
snmp-server enable traps snmp linkdown linkup
snmp-server enable traps eigrp
snmp-server host 10.100.0.50 version 2c netwatch_ro
