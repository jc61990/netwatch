Sites
Sites group your devices and control alert routing. Each site can have its own warning and critical email recipients — leave blank to use the global defaults from Email / SMTP.

SNMP profiles
Define named credential sets. Assign profiles to devices in the Devices tab. If a device has a v3 profile as primary and a v2c profile as fallback, v3 is tried first and v2c is used automatically if v3 times out.
Cisco IOS config reference
ip access-list standard ACL_SNMP_NETWATCH
  permit <monitoring-host-ip>
  deny   any log

! v2c
snmp-server community <community> ro ACL_SNMP_NETWATCH

! v3
snmp-server group NETWATCH_GROUP v3 priv
snmp-server user <username> NETWATCH_GROUP v3 auth sha <auth-pass> priv aes 128 <priv-pass>

snmp-server enable traps snmp linkdown linkup
snmp-server enable traps eigrp
snmp-server host <monitoring-host-ip> version 2c <community>

How long Prometheus keeps data on disk. Use 90d, 12w, 1y format. Changing this requires restarting the Prometheus systemd unit — the Reload button handles this automatically.
