Monitored devices
One row per Cisco device. Assign a primary SNMP profile and optional fallback. If the primary (v3) times out the fallback (v2c) is tried automatically. Click Test to verify SNMP reachability from the monitoring host.
IPSiteHostnamePrimary SNMPFallbackTest
SNMP profiles
Named credential sets assigned per device. Add both a v3 and v2c profile to enable automatic fallback.
Cisco IOS config reference
ip access-list standard ACL_SNMP_NETWATCH
  permit <monitoring-host-ip>
  deny   any log

! v2c
snmp-server community <community> ro ACL_SNMP_NETWATCH

! v3
snmp-server group NETWATCH_GROUP v3 priv
snmp-server user <user> NETWATCH_GROUP v3 auth sha <auth-pass> priv aes 128 <priv-pass>

snmp-server enable traps snmp linkdown linkup
snmp-server enable traps eigrp
